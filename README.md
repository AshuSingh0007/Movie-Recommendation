# Movie Recommendation System

This is a Python-based Movie Recommendation System built using machine learning and Streamlit for the user interface.

## Dataset Source
This project uses the IMDB 5000 Movie Dataset available on Kaggle.

## Features
- Simple and responsive UI
- Shows movie story/overview
- Displays movie poster
- Provides director and cast information
- Shows total votes and IMDB ratings
- Generates recommendations based on:
  - Similar movies
  - Movie genres

## Project Structure
- `App.py`: Main Streamlit application file
- `Classifier.py`: Contains the KNN-based recommendation algorithm
- `Movie_Data_Processing.ipynb`: Jupyter notebook for preprocessing and analysis
- `requirements.txt`: List of required Python libraries
